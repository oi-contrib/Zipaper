defineRouter
始于 v0.1.0

引入 

import { defineRouter } from "zipaper";

使用

比如： 

defineRouter({
    routers: [{
        path: "/",
        redirect: "/page1"
    }, {
        path: "/page1",
        component: () => import("./pages/page1/index.js"),
        meta: {
            title: "页面一"
        }
    }, {
        path: "/page2",
        component: () => import("./pages/page2/index.js"),
    }]
})

定义好路由以后，直接在需要路由切换的地方添加（支持多级路由，通过children属性配置子路由）： 

<router></router>

跳转

代码中触发跳转的例子： 

this.$goto("/page1")

元数据

比如上面定义的meta就是元数据，获取方式： 

this._router.meta.title
