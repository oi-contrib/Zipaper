defineDirective
始于 v0.1.0

引入 

import { defineDirective } from "zipaper";

使用 

let directive = defineDirective(option);

其中option是一个JSON，用于定义指令具体行为。

created

指令所在元素创建好后执行： 

option = {
    created(el: HTMLElement, binding) => void
}

binding格式如下（下同）： 

binding = {
    type: string, // 比如z-on:click就是"click"
    target: Zipaper,
    value: any, // 比如z-bind="val1"就是this.val1的值
    exp: string, // 比如z-bind="val1"就是"val1"
}

update

数据更新的时候触发： 

option = {
    update(el: HTMLElement, binding) => void
}
