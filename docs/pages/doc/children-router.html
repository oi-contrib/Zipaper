如何配置多级路由？

一级路由

配置一级路由非常简单，比如： 

defineRouter({
    routers:[{
        path: "/xxx1",
        component: () => import("./pages/xxx1/index.js"),
    },
    ......]
})

同时，需要在页面添加如下代码来表示当前路由xxx1的渲染位置： 

<router></router>

此时的路由类似这样：#/xxx1。如果主动跳转，直接： 

this.$goto("/xxx1")

二级路由

首先，我们需要在存在二级路由的地方添加子路由配置，比如： 

defineRouter({
    routers:[{
        path: "/xxx1",
        component: () => import("./pages/xxx1/index.js"),
        children:[{
            path: "/xxx2",
            component: () => import("./pages/xxx2/index.js"),
        },......]
    },
    ......]
})

因为上面我们是在一级路由xxx1下添加的二级路由xxx2，所以还需要在路由xxx1对应的页面中添加router标签来确定xxx2的渲染位置： 

<router></router>

此时的路由类似这样：#/xxx1/xxx2。如果主动跳转，直接： 

this.$goto("/xxx1/xxx2")

多级路由类似，在需要的父路由添加子路由即可，理论上可以有任意层。
