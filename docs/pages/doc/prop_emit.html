父子之间数据传递

父传递给子组件

我们可以在定义组件的时候，配置props： 

defineElement({
    props: {
        info: {
            type: String,
            required: true,
        }
    }
})

然后使用的时候，直接传递： 

<ui-demo info="来自父组件的数据"></ui-demo>

在组件内部，就可以获取到这个数据： 

this._props.info

子传递给父组件

依赖事件触发的方式，所以定义组件的时候，需要定义好事件： 

defineElement({
    emits: ["eventName"]
})

使用组件的时候，直接： 

<ui-demo z-on:eventName="doit"></ui-demo>

具体doit定义如下： 

doit(event){
    // event.data 就是来自子组件的数据
}

而在子组件中，需要传递给父组件数据的时候，只需要： 

this.$emit("eventName", "来自子组件的数据")
